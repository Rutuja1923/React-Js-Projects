 # React App (Vite)

## Installation  
1. Install Node.js (if not installed)  

2. Create a React app:  
```sh
   npm create vite@latest my-app 
```

3. Navigate to the project:
```sh
    cd my-app
```

4. Install dependencies:
```sh
    npm install
```

## Running the Project
```sh
    npm run dev
```

The app runs at the URL shown in the terminal (e.g., http://localhost:5173/)
